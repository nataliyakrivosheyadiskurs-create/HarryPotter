⚙ Настройка Firebase

Для синхронизации нужен Firebase. Вставьте конфигурацию из вашего проекта.

📜 Инструкция
  1. Откройте console.firebase.google.com
  2. Создайте проект (бесплатно)
  3. Добавьте Web-приложение (</>)
  4. Скопируйте конфигурацию
  5. Перейдите в Realtime Database → Создать базу → Тестовый режим

🖼 Картинки персонажей

Папка Google Drive с картинками. Файлы: 001.jpg, 002.jpg... до 120.jpg

💡 Папка должна быть открыта для всех. Формат URL смотрите в инструкции ниже.

Вариант 1 — прямые ссылки:
Загрузите картинки в Firebase Storage или любой хостинг.
URL будет вида: https://example.com/cards/
Файлы: 001.jpg, 002.jpg и т.д.

Вариант 2 — Google Drive:
1. Загрузите файлы с именами 001.jpg...120.jpg в папку
2. Откройте общий доступ к папке → Все у кого есть ссылка
3. Для каждого файла скопируйте ID из ссылки
4. URL = https://drive.google.com/uc?export=view&id=FILE_ID

⚠️ Google Drive неудобен для 120 файлов. Рекомендуем Firebase Storage — там можно задать базовый URL и подставлять 001.jpg, 002.jpg автоматически.
